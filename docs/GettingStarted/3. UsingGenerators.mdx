## What's a generator?

One of the most powerful features of `react-hexgrid` is its generator algorithms - helper functions
that can create patterns of hexagons for you.  

## Usage

Generators return an array of hexagon coordinates (q, r, s coordinates).  You can use them like 
any other array!  Right now, you can make a ring, a spiral, a rectangle, a parallelogram (oriented rectangle) 
and a hexagon.
Here's a basic example, using the hexagon-shaped generator:

```tsx
import { Hexagon, HexGrid, HexagonGenerator } from '@7dir/react-hexgrid';
export default function App() {
    return(
        <HexGrid width="100%">
                {GridGenerator.hexagon(2).map((hex, i) => (
                    <Hexagon key={i} q={hex.q} r={hex.r} s={hex.s} className="tutorial" />
                ))}
        </HexGrid>
    )
}
```

(using the same `App.css` file from earlier)
