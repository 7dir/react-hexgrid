# react-hexgrid

`react-hexgrid` helps you create and maintain regular grids of hexagons without the headache of wrangling with SVGs.  With native
support of React hooks like `onMouseEnter` and `onMouseLeave`, it's simple to make your hexagons interactive.

## Why should you use it?

`react-hexgrid` boasts 0 dependencies and full TypeScript support, with ever single function and component strictly and accurately typed.
Combined with transparent Storybook visual tests, you can be confident that any code you write will be lasting and maintainable.
